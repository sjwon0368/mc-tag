마인크래프트 Paper 1.21.5 서버에 사용 가능한 술래잡기 플러그인.
A Minecraft Server tag game plugin(Paper 1.21.5)

## 사용 방법
### 명령어(Usage)
`/tagstart <술래> <도망자>`
> 예시: `/tagstart user1 user2`
